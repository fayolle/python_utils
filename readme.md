# Python utils

A collection of Python utils, mostly for geometry processing.
Rely on NumPy and SciPy.


## Installation
Create a Python virtualenv
```bash
python -m venv venv
```
activate on Windows
```bash
./venv/Scripts/Activate
```
or OSX/Linux
```bash
./venv/bin/activate
```
and install the requirements
```bash
pip install -r requirements.txt
```
